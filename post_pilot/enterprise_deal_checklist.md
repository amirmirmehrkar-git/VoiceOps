# Enterprise Deal Checklist

## Pre-Engagement

### Qualification
- [ ] Enterprise size confirmed (1000+ employees or $X+ revenue)
- [ ] Budget confirmed ($X+ available)
- [ ] Decision maker identified
- [ ] Timeline established
- [ ] Use case validated

### Requirements Gathering
- [ ] Technical requirements documented
- [ ] Integration requirements identified
- [ ] Security requirements collected
- [ ] Compliance requirements understood
- [ ] Custom features requested

---

## Proposal Phase

### Documentation
- [ ] Custom proposal prepared
- [ ] Pricing proposal sent
- [ ] Technical architecture reviewed
- [ ] Security documentation provided
- [ ] Legal documents prepared (MSA, DPA, Order Form)

### Stakeholder Alignment
- [ ] Technical team engaged
- [ ] Security team engaged
- [ ] Legal/procurement engaged
- [ ] Executive sponsor identified
- [ ] Champion identified

---

## Negotiation Phase

### Pricing & Terms
- [ ] Pricing negotiated
- [ ] Contract terms agreed
- [ ] SLA defined
- [ ] Payment terms agreed
- [ ] Renewal terms set

### Legal
- [ ] MSA reviewed and approved
- [ ] DPA signed
- [ ] Order Form completed
- [ ] Custom terms negotiated (if needed)
- [ ] Insurance requirements met

### Security & Compliance
- [ ] Security review completed
- [ ] Vendor questionnaire answered
- [ ] SOC2 report shared (if available)
- [ ] Penetration test results shared
- [ ] Compliance requirements met

---

## Implementation Planning

### Technical
- [ ] Integration plan created
- [ ] Custom development scoped
- [ ] Timeline established
- [ ] Resource allocation confirmed
- [ ] Success criteria defined

### Onboarding
- [ ] Onboarding plan created
- [ ] Training schedule set
- [ ] Support plan defined
- [ ] Account manager assigned
- [ ] Kickoff meeting scheduled

---

## Closing

### Final Steps
- [ ] All documents signed
- [ ] Payment method confirmed
- [ ] Account activated
- [ ] Access granted
- [ ] Welcome email sent

### Post-Close
- [ ] Implementation kickoff
- [ ] Regular check-ins scheduled
- [ ] Success metrics tracking
- [ ] Expansion opportunities identified

---

## Red Flags (Watch For)

- [ ] Unclear decision process
- [ ] No budget allocated
- [ ] Timeline too aggressive
- [ ] Unrealistic requirements
- [ ] Multiple stakeholders not aligned

---

## Success Criteria

- [ ] Contract signed
- [ ] Payment received
- [ ] Implementation started
- [ ] Customer satisfied
- [ ] Expansion opportunities identified

---

## Timeline

**Typical Enterprise Deal**: 3-6 months
- Qualification: 2-4 weeks
- Proposal: 2-4 weeks
- Negotiation: 4-8 weeks
- Legal/Procurement: 4-8 weeks
- Closing: 1-2 weeks
